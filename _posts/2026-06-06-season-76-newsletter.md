Welcome to Season 76! The Dominion Online website has some new features that should make setting up and viewing League matches easier. Let's have a look!

**League Table Setting**

When creating a table, there is now a selection called "League Match" that should be selected for playing your matches. Try to make selecting this button a habit - in future seasons, we may not count games played without this setting enabled. See its location in the table setup screen below:

<img src="/img/faq/LeagueMatchButton.png" height="400">

This setting will:
- Automatically alternate starts between games
- Hide spectator chat as soon as the first game starts
- After each game post a match summary in the chat
- Fix table settings to match league rules
- Assist with reloading games if necessary
- Add the table to a special section at the bottom of the Friends section of the Automatch page

There are additional changes and features planned for this setting. If you have a suggestion, please post it in the #league or #2nd-shuffle-client channels of the Dominion Discord.

**Timer Update**

Thank you to those who participated in our timer Town Hall last season. With that discussion and poll responses in mind, we are aiming to move to a timer for our play speed enforcement starting in one or two seasons, with the seasons between now and then being used for further timer parameter refinement to make sure we have a setting appropriate for League.

This season, the "League Match" setting will put in place a timer with more time given back each turn compared to Patient. The website developers have generously agreed to track additional data regarding how much of this timer gets used that we will use to inform us what further changes could be needed. For this season, this timer will continue to have no consequences and players may not lose games due to running out of time; that said, the AET system will still be enforced.

Best of luck to all and enjoy the season!
